# Think-nix-way / Dotpad

My Config NixOS declarative optimaze for display **ThinkPad 14 inch**. 
Before migrate from Arch Hyprland to system than stable, reproduction, and using *scroll-tiling* from Niri.

### Tech Stack:
- **OS:** NixOS (Unstable/Stable)
- **Compositor:** Niri (Scrollable tiling)
- **Terminal:** Kitty (with image preview support)
- **File Manager:** Yazi & Ranger
- **Hardware:** Lenovo ThinkPad (14")

### Steps use:
1. Copy `configuration.nix` to `/etc/nixos/`
2. Running `sudo nixos-rebuild switch`
3. done, maybe!
